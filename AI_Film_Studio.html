🍌
Nano Banana Pro — 12-Angle Character Sheet
Generate a complete character reference from one master image
  1. Create a single hero image in ChatGPT using the "Nano Banana Pro style" — photorealistic, sharp, neutral background, premium commercial look.
  2. Open Nano Banana Pro GPT — paste your image and request "Angles 2.0: generate 12 angles" for a full character sheet.
  3. Save 3–4 of the best angles into your Reference Assets tab as the character's visual bible.
  4. Use those angle images as input references in Kling Elements for locked character consistency across all clips.
⬡
Kling 3.0 Elements — Character Consistency
Lock your subject across all video clips
  1. Upload your Nano Banana character image into Kling's Elements feature (not just the Image-to-Video input).
  2. Set the Element weight to 0.7–0.85. Too high = stiff; too low = ignores it.
  3. Write your video prompt normally — the Element handles face/body consistency so you can focus on action and environment.
  4. For multi-shot sequences: keep the same Element across all clips and only vary the camera angle and action.
Pro tip: Use a 3/4 angle (not full front) as your Element reference — it gives the model more spatial information and produces more natural-looking motion.

The realism stack — use these together for cinematic output
  1. In Higgsfield, select model: Soul 2 (custom Nano Banana 2 variant — optimized for cinematic texture and skin realism).

Workflow guide

How Studio S works

The Supergood AI film pipeline

Four stages. Each with a specialist tool. AI Film Studio sits at the centre — prompts to stills to video. Human craft on top.

Stage 1

Claude — ideate & plan

Drop your brief, references, and any existing scripts into an assets folder. Ask Claude to read them and write project-plan.md — concepts, shot list, and production notes. Studio S reads the plan and scaffolds your project automatically.

Pro tip: Test your assets folder with Claude in a normal chat first. Once the plan looks right, drop it into Studio S.

Stage 2 — you are here

Studio S — prompts to stills to video

1

Build assets first. Generate character sheets (gray bg, one face per panel), location refs at 3/4 angle. Test each in motion before locking.

2

Set the global style prefix in Multi-Shot. One edit updates every shot's compiled prompt. Use per-shot overrides only when a scene genuinely differs.

3

Choreograph motion explicitly. "He dances" generates mush. Spell out each move by name. Sync to the music beat.

4

The final cut is the best seconds of many tries. Iteration is the skill. Run multiples, pull the keepers, edit tight.

On quality degradation: each OmniEdit pass softens the original. Composite the source + edit in Finishing — face/skin from original, change from the edit layer only.

Stage 3

Palmier — AI assembly

Write a kickoff prompt telling Claude what to do with your clips — scene order, pacing, beat sync, handle lengths. Claude edits the Palmier timeline directly via MCP. The finished timeline exports as NLE XML or ProRes.

Models in Palmier: Kling 3.0, Seedance, and Nano Banana Pro run natively on the timeline. Regenerate any clip in-place without leaving the project.

Stage 4

Finishing — human craft

Premiere for narrative cut and SFX. DaVinci to grade to the cinematographer look. After Effects for motion graphics. The AI built the raw material; finishing is where the quality ceiling goes up.

Hand this project off to a Claude + Palmier Pro session, where Claude writes the prompts (using this app's doctrine), generates the stills/clips/audio, and assembles the timeline — shot by shot. This is the AI edit stage, before the human finishing crafts.
