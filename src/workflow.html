How Studio S works

The Supergood AI film pipeline

Four stages. Each with a specialist tool. Studio S sits at the centre — prompts to stills to video. Human craft on top.

Stage 1

Claude — ideate & plan

Drop your brief, references, and any existing scripts into an assets folder. Ask Claude to read them and write project-plan.md — concepts, shot list, and production notes. Studio S reads the plan and scaffolds your project automatically.

Pro tip: Test your assets folder with Claude in a normal chat first. Once the plan looks right, drop it into Studio S.

Stage 2 — you are here

Studio S — prompts to stills to video

1

Build assets first. On the References page, run the wizard per character, location, prop and look — it compiles a generation-ready prompt. Generate it in your image tool, attach the result to the card, then pick those references inside each shot's prompt. Test each in motion before locking.

2

Set the global style prefix in Multi-Shot. One edit updates every shot's compiled prompt. Use per-shot overrides only when a scene genuinely differs.

3

Choreograph motion explicitly. "He dances" generates mush. Spell out each move by name. Sync to the music beat.

4

The final cut is the best seconds of many tries. Iteration is the skill. Run multiples, pull the keepers, edit tight.

On quality degradation: each OmniEdit pass softens the original. Composite the source + edit in Finishing — face/skin from original, change from the edit layer only.

Stage 3

Palmier — AI assembly

Write a kickoff prompt telling Claude what to do with your clips — scene order, pacing, beat sync, handle lengths. Claude edits the Palmier timeline directly via MCP. The finished timeline exports as NLE XML or ProRes.

Models in Palmier: Kling 3.0, Seedance, and Nano Banana Pro run natively on the timeline. Regenerate any clip in-place without leaving the project.

Stage 4

Finishing — human craft

Premiere for narrative cut and SFX. DaVinci to grade to the cinematographer look. After Effects for motion graphics. The AI built the raw material; finishing is where the quality ceiling goes up.
